# ci-cd boilerplate

GitHub Actions CI/CD workflows for Node.js/TypeScript projects.

## Included workflows

- **ci.yml** — lint, test, build on every PR
- **deploy.yml** — deploy to VPS/server on push to main
- **release.yml** — semantic versioning + changelog

"Scaffold a ci-cd project in ./.github"
