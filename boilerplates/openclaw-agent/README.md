# openclaw-agent boilerplate

A production-ready OpenClaw agent workspace starter.

## What's included

- `AGENTS.md` — workspace rules and conventions
- `SOUL.md` — agent identity and persona
- `USER.md` — user profile
- `MEMORY.md` — long-term memory template
- `HEARTBEAT.md` — periodic task config
- `TOOLS.md` — tool/infra notes
- `memory/` — directory for daily session logs

## Getting started

1. Edit `SOUL.md` — define your agent's name, role, and behavior
2. Edit `USER.md` — fill in your human's details
3. Point your OpenClaw agent at this workspace directory

```
"Scaffold an openclaw-agent project in ./my-agent"
```
