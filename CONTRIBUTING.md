# Contributing to boilerforge

## Adding a boilerplate

1. Fork the repo
2. Create a folder under `boilerplates/your-boilerplate-name/`
3. Add all template files
4. Add a `README.md` inside explaining what it is and how to use it
5. Update the table in the root `README.md`
6. Open a PR

## The bar for a boilerplate

- **Production-ready** — not a tutorial, not a toy
- **Opinionated** — make choices, don't leave everything configurable
- **Maintained** — you're committing to keeping it updated
- **Documented** — clear README, clear purpose

## Boilerplate structure

```
boilerplates/
└── your-boilerplate-name/
    ├── README.md        # Required
    ├── ...              # All template files
```
